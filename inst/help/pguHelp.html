Import

Manages the upload of the upload of the dataset. The data needs to be of type .xlsx and must comprise the sheets: raw_data, loq, metadata.

Browse

Upload a file to the R server.

Import

Import the uploaded file for analysis.

raw data

Specifies which data types were recognized from the imported data sheet raw_data.

loq

Specifies which data types were recognized from the imported data sheet loq.

metadata

Specifies which data types were recognized from the imported data sheet metadata.

Filter

Filter the raw dataset. All further analysis steps will be performed on the filtered dataset.

Filter

Creates a new dataset comprising all selected data.

Filter inverse

Creates a new dataset comprising all columna that are not selected. The row selection will remain.

Reset

Resets filter parameter. The complete data set is selected.

Statistics

Displays some statistical parameters about the filtered data set.

Missings

Displays the amount of missing entrys within each selected column.

Explore

Examines the data space of the filtered data set graphically.

Abscissa

Select the column that will serve as abscissa within the graphical exploration.

Ordinate

Select the column that will serve as ordinate within the graphical exploration.

Abscissa Statistics

Graphical exploration of the abscissa attribute distribution displayed as box-plot and histogram. Below the plot, some statistical parameters are displayed. Will only work for abscissa attributes of type numeric. Otherwise it will be blank.

Ordinate Statistics

Graphical exploration of the ordinate attribute distribution displayed as box-plot and histogram. Below the plot, some statistical parameters are displayed. Will only work for ordinate attributes of type numeric. Otherwise it will be blank.

LOQ Detect

Identify candidates that exist outside of the user provided LOQ intervals

analyze LOQ

Runs the analysis. Based on the loq data sheet from the imported data each value is testet against the provided LOQ interval limits. A list of all candidates that exist outside of the user provided LOQ intervals is crerated.

Reset

Resets the analysis parameters so that they match those of the analysis performed.

LOQ Distribution

Graphical exploration of the amount of candidates per attribute that exist outside of the user provided LOQ intervals. In order to distinguish between candidates that exist below the LLOQ and canditates that exist above the ULOQ the results are color coded.

NA handling

Defines how NA values should be handled during analysis.

Feature

Choose a feature for further detailed exploration.

Feature Plot

Graphical exploration of the selected feature's candidate distribution displayed as box-plot and histogram. The LOQs are indicated as dotted lines.

Feature Data

List of all candidates from the selected attribute. LOQ outlier candidates are highlighted by their background color.

LOQ Statistics

Statistical composition of the LOQ outlier candidate distribution.

LOQ Outlier

List of the detected LOQ outlier candidates. The respective critical values are marked by their background color.

LOQ Data

The entire filtered data set with each LOQ outlier candidate highlighted by the background color respectively.

LOQ Mutate

Substitute the detected LOQ outlier candidates with specific values.

Runs the analysis. Based on the loq data sheet from the imported data each LOQ outlier candidate is substituted by a distict user defined value. The resulting data set is passed to further analysis.

Reset

Resets the analysis parameters so that they match those of the analysis performed.

LOQ Distribution

Graphical exploration of the amount of candidates per attribute that exist outside of the user provided LOQ intervals. In order to distinguish between candidates that exist below the LLOQ and canditates that exist above the ULOQ the results are color coded.

LLOQ Substitute

Defines the substitute for candidates who are below the LLOQ.

ULOQ Substitute

Defines the substitute for candidates who are above the ULOQ.

Choose a feature for further detailed exploration.

Feature Plot

Graphical exploration of the selected feature's candidate distribution displayed as box-plot and histogram. The LOQs are indicated as dotted lines.

. Substituted candidates are color coded.

List of all candidates from the selected attribute. Substituted LOQ outlier candidates are highlighted by their background color.

LOQ Statistics

Statistical composition of the LOQ outlier candidate distribution.

LOQ Outlier

List of the substituted LOQ outlier candidates. The respective critical values are marked by their background color.

LOQ Data

The entire filtered data set with each substituted LOQ outlier candidate highlighted by the background color respectively.

Transformation Detect

Performes multiple user defiend data transformations. The resulting transforations are tested against a normal distribution.

Detect

Runs the comparison analysis of the user defined transfromation types.

Reset

Resets the analysis parameters so that they match those of the analysis performed.

Optimal Transformations

Specifies the data transformation that led to the minimum deviation from a normal distribution as measured by the respective test parameter.

Optimal Transformation Parameters

The parameters, which belong to the optimal transformations.

Transformation Mutate

Performes a data transformation and provides multiple statistical hypothesis tests for normality (Shapiro-Wilk test, Kolmogorov-Smirnov test, Anderson-Darling test). Data are further Z-score transfromed, before passed to the next analysis step.

set globally

Performes the same data transformation for all attributes. Needs to be run first.

set

Changes the data transformation type for individual user defined attributes.

Reset

Resets the analysis parameters so that they match those of the analysis performed.

Transfromation Type

The actual chosen data transformation type for the user selected feature.

Mirror Data

If activatetd, the data is reflected through the origin before transformation.

Feature

Select an attribute for a detailed graphical distribution exploration and eventually changing its type of transformation.

Feature Transformation

Graphical exploration of the deviation of the selected transfromed attribute's distribution from a normal distribution. The blue line indicates the model, that fits the attribute distribution best. The upper left shows a histogram, the upper right a qq-plot, the lower left a residual plot, and the lower right the residual distribution compared to the expected value from the model.

Model Parameter

Represents the model parameter for the user defined transformed attribute.

Model Quality

List of results of all statistical hypothesis tests performed against normal distribution of the transformed attribute.

Global Transfromation Parameter

The actual user defined transformation parameter for each attribute.

Global Model Parameter

Table of model parameters for the whole filtered dataset.

Global Model Quality

Result of the model fitting procedure.

Global Test Results

List of results of all statistical hypothesis tests performed against normal distribution of all transformed attributes.

Transfromed Data

The normalized and Z-score transfromed data set that is passed to further analysis.

Imputation Detect

Detect imputation sites and explore theri distribution throughout the data set.

Imputation Site Heatmap

Graphical exploration of the imputation site distribution throughout the data set. The histogram on the left shows fraction of missing values per attribute in descending order. The right shows a heatmap of imputation site occurence patterns in ascending order.

Imputation Site Statistics

Quantitative numerical representation of the of imputation site occurence patterns.

Imputation Site Setails

A list of all candidates that have missing values in at least one attribute.

Imputation Site Data

The normalized and Z-score transfromed data set that is passed to further analysis.

Imputation Mutate

Replace missing values with substitutes by classical and AI-powerd imputation algorithms.

Mutate Imputation Sites

Run selected imputation algorithm with current settings.

Reset Imputation Settings

Resets parameters so that they match those of performed processing routine.

Mutation Method

Select an imputation algorithm: median, mean, mu, mc, knn, pmm, CART, rf, M5P, amelia, amelia_bound.

Seed

Set the seed for the selected imputation algorithm.

Iterations

Select the number of times, the analysis is run. The result with the least deviation copared to the original data set is selected at the end.

Feature

Select an attribute for a detailed graphical exploration of the results of the performed imputation procedure.

Feature Plot

Graphical exploration of the selected feature's candidate distribution displayed as box-plot and histogram. The imputation sides are color coded.

Feature Detail

Numerical exploration of the selected feature's imputation site substitutes.

Imputation Site Detail

List of all candidates with substituted imputation sites.

Imputation Site Data

The normalized and Z-score transfromed data set with substituted imputation sites that is passed to further analysis.

Outliers Detect

Detect Outlier

Runs the analysis.

Outlier Distribution

Graphical exploration of the distribution of the occurrence of outliers in the data set.

Outlier Statistics

Quantitative statistical evaluation of the occurrence of outliers in the data set.

Outlier Details

Detailed list of candidtates with at least one outlier.

Outlier Data

The normalized and Z-score transfromed data set with substituted imputation sites that is passed to further analysis.

Outliers Mutate

Replace missing values with substitutes by classical and AI-powerd substitution algorithms. For this purpose outliers are handled as imputation sites.

Mutate Imputation Sites

Run selected imputation algorithm with current settings.

Reset Imputation Settings

Resets parameters so that they match those of performed processing routine.

Seed

Set the seed for the selected imputation algorithm.

Iterations

Select the number of times, the analysis is run. The result with the least deviation copared to the original data set is selected at the end.

Feature

Select an attribute for a detailed graphical exploration of the results of the performed imputation procedure.

Feature Plot

Graphical exploration of the selected feature's candidate distribution displayed as box-plot and histogram. The imputation sides are color coded.

Feature Detail

Numerical exploration of the selected feature's imputation site substitutes.

Outliers Mutated Details

List of all candidates with substituted imputation sites.

Outlier Mutated Data

The normalized and Z-score transfromed data set with substituted imputation sites and substituted outliers that is passed to further analysis.

Analysis Correlation

Performes a correlation analysis on the cleaned, normalized and Z-score transfromed data set.

Calculate

Run analysis.

Pearson

Matrices comprising the pair wise Pearson's correlation coefficients and the respective p-value.

Spearman

Matrices comprising the pair wise Spearman's rank correlation coefficients and the respective p-value.

Kendall

Matrices comprising the pair wise Kendall's rank correlation coefficients and the respective p-value.

Analysis Regression

Performes linear regression analysis (To be more precise a robust regression) on the cleaned, normalized and Z-score transfromed data set

Calculate

Run analysis.

Abscissa

Select the column that will serve as abscissa within the graphical exploration.

Ordinate

Select the column that will serve as ordinate within the graphical exploration.

Model

Graphical and numerical exploration of the robust regression analysis of the selected abscissa and ordinate attributes.

Intercept

Matrices comprising the pair wise intercepts of the regression models and their respective p-values.

Slope

Matrices comprising the pair wise slope of the regression models and their respective p-values.

Export

Exports the analysis results in Excel format

Perfomres the download. The option is only available if a normalized and Z-score transfromed data set with substituted imputation sites and substituted outliers is present.
